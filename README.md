# 🚀 Multimodal Sentiment & Emotion Analysis AI

This project performs:

- 📝 Text Sentiment Analysis (DistilBERT)
- 📷 Face Emotion Detection (FER + OpenCV)
- 🎤 Voice Sentiment Analysis (SpeechRecognition + NLP)
- 📊 CSV Customer Dashboard (Plotly)

## Tech Stack
- Python
- Streamlit
- Transformers
- FER
- OpenCV
- Librosa
- Plotly

## Deployment
- Local via Streamlit
- Public via ngrok
- Ready for HuggingFace Spaces
